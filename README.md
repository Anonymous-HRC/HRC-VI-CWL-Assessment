# HRC-VI-CWL-Assessment

Anonymous supplementary project page for:

**Vision-based Cognitive Workload Assessment in Human-Robot Collaborative Assembly**

This repository is a static GitHub Pages site. It uses only relative paths and
does not require a build step.

## Structure

```text
/
├── index.html
├── assets/
│   ├── images/
│   ├── videos/
│   └── figures/
├── css/
├── js/
└── README.md
```

## GitHub Pages

Use GitHub Pages with:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

After Pages is enabled, the site should be available at:

```text
https://anonymous-hrc.github.io/HRC-VI-CWL-Assessment/
```

## Anonymous Review Checklist

- Keep author names as `Anonymous Authors`.
- Avoid personal GitHub usernames, institutional identifiers, and personal URLs.
- Check PDFs, videos, figures, filenames, and metadata before submission.
- Replace placeholder figures and text only with anonymized materials.
